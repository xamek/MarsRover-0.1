# MarsRover-0.1

a solution to Mars Rover kata

to run the application you need node installed 

run:
```
npm install
```
to run tests:
```
npm run test
```
to run API server:
```
npm run start
```
use this url to test API server:
http://localhost:8080/deployAndMove?start=4.2&direction=EAST&command=FLFFFRFLB
